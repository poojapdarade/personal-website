Important things to know to write Javascript

  1. First thing we are going to learn is comments in the javascript
    // is used for in-line comments
    /* */ is used for multi-line comment
    You end every line in Javascript with ; semicolon
  2. Data types
    undefined: Something which has not been defined yet
    null: When you set this property to an element that results nothing for the output for that element
    boolean: True or False
    string : set of characters
    symbol: A symbol is an immutable primitive value that is unique
    number: numbers
    object: An object can store a lot of different key value pairs.
  3. Variables:
    We often set data into a variable. A variable allows computers to store and manipulate data in a dynamic fashion. It is basically a label to point to a data. To declare variable
    There are three ways to declare
    var muName = "Beau" followed by myName = 8
    let ourName = "Pooja"
    const yourName = "John"
    A difference between var,const and let. A var is going to be able to be used throughout your whole program. Let will only be able to use within scope where you declare that and const is a variable that can never change. There is difference between declaring the variable and assigning the variable.
    var a; This how to declare variable.
    var b=2; By using = assignment operator we can assign variable to a value.
